PostgreSQL 数据库 SKILLs:

- pg-awr-report : 生成 AWR 报告
- pg-find-bloat : 分析膨胀表、索引
- pg-find-unused-index : 分析未使用索引
- pg-load-spike-forensics : 对给定的一段可疑时间窗口做数据库负载飙升的多维取证分析
- pg-parameter-tuning-advisor : 分析数据库工作负载和环境, 给出参数优化建议
- pg-security-audit : 数据库安全评估, 给出建议
- pg-sql-tuning-advisor : 分析数据库工作负载和环境, 对指定 SQL 给出优化建议
- pg-log-analyzer : 分析数据库给定时间段内的日志, 给出分析建议
- pg-design-audit : 分析数据库元数据和采样, 根据设计规范给出审查指引建议
- pg-top-sql-analyze : 分析数据库最近时间段各个维度的 TOP SQL, 给出优化建议
- pg-bloat-root-cause : 分析数据库存在的潜在膨胀风险, 给出建议
- pg-large-table-optimize : 分析数据库大表, 以及大表的工作负载风格, 根据工作负载风格给出优化建议
